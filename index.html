Praktikugens arbejdsplan

  • Mandag: Hvad er en hjemmeside? - Otto forstår hvad HTML og CSS er, har sit udviklingsmiljø klar, og har en fungerende side i browseren.
  • Tirsdag: Layout og struktur - Otto lærer semantisk HTML, flexbox til layout, og bygger flere sider. Han oplever irritationen ved gentagen markup – og er klar til løsningen i morgen.
  • Onsdag: Eleventy og komponenter - Otto lærer at strukturere CSS i flere filer, og migrerer projektet til Eleventy så header og footer kun skal vedligeholdes ét sted. Han opretter de første blogindlæg.
  • Torsdag: Collections, JavaScript og go live - Otto lærer Eleventy collections med mappestruktur så blogindlæg automatisk listes efter kategori, skriver i Markdown, tilføjer JavaScript med en burgermenu, og får siden live på Vercel. Til sidst lidt tid til finpudsning af CSS.
  • Fredag: Afslutning - Finpudsning og afslutning af praktikugen.
  • Videre med Barcelona Bloggen - Her er nogle ting du kan arbejde videre med på egen hånd. Tag én ting ad gangen – der er ingen grund til at skynde sig.
